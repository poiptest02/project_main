Pet Mode

Pet Mode Precondition

  • EOL.POWER_TYPE : EV(전기차 전용)
  • REGION : KR(현재 KR 서버만 계정 연동 가능)
  • 프로필 계정 : CCS 서버와 연동 완료 상태일 것

기본적인 펫 모드 실행 워크 플로우 확인 방법

  1. Precondtion 진행 완료 상태
  2. AVNT 의 CCU 해제
  3. 프레임 시그널 값
    VPC_01_10ms VPC_PetModRdySta 0x1
    CCU_CCS_08_300ms PetModeOpt 0x1
    신호 인가
  4. Vehicle Settings 앱 > Convenience 사이드 메뉴 > 펫 모드 퀵버튼 표시 여부 확인
  5. 프레임 시그널 값
    CCU_CCS_08_300ms PetModeSta 0x1
    신호 인가
  6. 펫 모드 퀵 버튼 > Activate 버튼 실행 > 펫 모드 가이드 팝업 표시 여부 확인
  7. 펫 모드 가이드 팝업의 OK 버튼 실행 > Pet Mode Ready Status 팝업 표시 여부 확인
  8. 프레임 시그널 값
    CCU_CCS_08_300ms PetModeRes 0x1
    CCU_CCS_08_300ms PetModeSta 0x3
    신호 인가
  9. 펫 모드 화면으로 진입 확인

기본적인 펫 모드 종료 워크 플로우

  1. 프레임 시그널 값
    CCU_CCS_08_300ms PetModeSta 0x3
    CCU_CCS_08_300ms Petcare_DrLkUnLkSta 0x2
    CCU_CCS_08_300ms PetModeRes 0x2
    CCU_CCS_08_300ms PetModeFinish 0x1
    신호 인가
    • 펫 모드 화면 종료되고 관련 토스트 팝업 표시 됨
      • Pet Mode Ready Status 팝업 표시된 상태 유지됨
      • Pet Mode Ready Status 팝업에서 Cancel 버튼 실행 시 팝업 종료되어야 함

다양한 조건 확인 시 r4j 사양서 충분히 숙지 후 진행 할 것

  1. 예시
    • PetModeFinish = 0x2 값을 주어 펫 모드 종료 시
      • 비정상적인 종료 중 Climate off 진행됨
      • 그에 따라 발생하는 종료 팝업도 상이함
